Projects/Publications

  • neXt-Ray: Deep Learning on Bone X-Rays [code] [paper] Built and trained Residual Network (ResNet) to predict whether Bone X-Rays from upper-extremity regions are normal or abnormal, outperforming or matching radiologist performance on finger and humerus studies
  • Multi-Org Data Exchange [overview] Led the development of the Multi-Org data exchange feature found in C360DM, allowing customers to retrieve data records from all their different Salesforce orgs by just issuing one query to DFS. I spearheaded the E2E implementation of the feature, from the OData->SOQL query conversions to the intricacies of doing so in an authenticated manner. Multi-Org has been one of the most popular features in C360DM
  • Transformation Engine [overview] Contributed crucial features to the engine that drives the metadata-based conversion of data from one schema to another as part of C360DM. Advancements I provided to the transformation engine include the abilities to specify in the metadata (1) which data records to sort by particular fields before transforming (2) subtype polymorphism via casting (3) how to navigate relationships arcs in data models
